# NHL Playoff Tracker (Chrome extension)

Manifest V3 popup that loads playoff-position teams from NHL standings and shows **next game** or **last result & highlights** per selected club.

Uses public API only: `https://api-web.nhle.com`.

## Install (development)

1. Clone this repo.
2. Chrome → **Extensions** → **Developer mode** → **Load unpacked** → select this folder.

## Privacy

Fetch targets are `api-web.nhle.com` for schedules and standings; highlight links resolve on `nhl.com`.
